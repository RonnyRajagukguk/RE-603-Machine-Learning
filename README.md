# RE-603 Machine Learning

Repository ini berisi kumpulan tugas dan latihan dari mata kuliah **Machine Learning RE-603**.

Setiap folder berisi materi, notebook, dan dataset yang digunakan pada masing-masing minggu pembelajaran.

---

## Week List

* **Week 2 – Python Basics**
  Materi dasar Python untuk analisis data seperti penggunaan variabel, struktur data, serta manipulasi data menggunakan Python.

* **Week 3 – Exploratory Data Analysis (EDA)**
  Analisis eksplorasi data yang mencakup:

  * Data understanding
  * Data visualization
  * Outlier detection
  * Missing value handling
  * Feature encoding
